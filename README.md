# Menu Driven Calculator (Python)

This is a simple menu-driven calculator program written in Python.
It allows users to perform basic arithmetic operations using a loop-based menu system.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Continuous execution using while loop
- Input validation
- Division by zero handling

## Concepts Used
- while loop
- if-elif-else
- exception handling
- user input
- basic arithmetic operations

## How to Run
1. Clone the repository
2. Run the file using:
   python calculator.py
